# 💼 Piyush Chauhan's Personal Portfolio

Welcome to my personal developer portfolio website — a space where I showcase my skills, projects, experience, and journey as a Computer Science Engineering student passionate about web development, AI, and real-world problem-solving.

🌐 Live Demo: (https://piyush-chauhan-portfolio.netlify.app)

---

## 🧑‍💻 About Me

I am a B.Tech (CSE) student with a strong interest in full-stack development, data science, and hackathon-based innovation. I’ve built this portfolio from scratch using HTML, CSS to represent my technical journey and projects.

---

## 🚀 Features

- Home Page: A clean, modern introduction with links to various sections.
- About Page: Includes a brief about me, skills section with responsive cards, and online coding profiles (LeetCode & GFG).
- Projects Page: Displays a list of key projects with descriptions and technologies used.
- Experience Page: Highlights my hackathon participation and internship experience using styled cards.
- Contact Page: Contains my social handles for easy reach-out.
- Responsive Design: Mobile-friendly layout using CSS grid/flexbox.
- Custom Styling: Built with pure CSS, featuring linear gradients, grid layouts, and interactive hover effects.

---

## 🛠️ Tech Stack

- Frontend: HTML5, CSS3
- Styling Techniques: Flexbox, Grid Layout, Media Queries
- Deployment: Netlify
